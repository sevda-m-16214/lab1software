# OmniHome Smart Hub

## Patterns Implemented

- **Singleton** (Thread-safe via Holder Idiom)  
- **Abstract Factory** (Budget vs Luxury product families)  
- **Adapter** (Glorb legacy thermostat integration)  
- **Builder** (Automation routines)  
- **Prototype** (Device configuration cloning)  

---

##  How to Run

1. Compile the project.  
2. Run `Main.java`.  
3. Observe pattern demonstrations in the console output.
